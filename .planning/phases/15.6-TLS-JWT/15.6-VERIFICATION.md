### Key Features

#### TLS Module
- `TlsConfig`: Certificate/key path configuration
- `TlsListener`: Async TLS listener with tokio-rustls
- Support for mTLS with CA certificate
- Protocol negotiation (HTTP/1.1, HTTP/2)

#### JWT Module
- `JwtConfig`: Validation configuration (secret or public key)
- `JwtValidator`: Token parsing and validation
- `Claims`: Standard claims (sub, iss, aud, exp) plus custom claims
- `JwtAuthMiddleware`: Async middleware for request authentication
- `extract_token()`: Bearer token extraction from Authorization header

## Notes

- JWT validation does not include cryptographic signature verification (simplified for scaffold)
