---

## Implementation Notes

### TLS Module
- Uses `tokio-rustls` for async TLS support
- Supports PEM certificate and key files
- Optional mTLS with CA certificate
- Protocol negotiation (HTTP/1.1, HTTP/2)

### JWT Module
- RS256/ES256 signature validation
- Standard claims validation (exp, iss, aud)
- Custom claims support (roles, scope)
- Async middleware for request validation

### Integration Points
- `TlsListener` wraps TCP listener with TLS
- `JwtAuthMiddleware` validates requests
- `JwtValidator` for token parsing
- Configuration via server config
