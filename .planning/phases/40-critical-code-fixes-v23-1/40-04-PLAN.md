# Phase 40 Plan 04: Enforce Stub Architecture Load Policy (typed LoadError)

**Goal:** Convert `ModelLoader::load()` stub-rejection error path from
`candle_core::Error::msg(...)` string to a typed `LoadError::StubNotAllowed`
variant. The `allow_stub` capability gate at `builder.rs:212-225` already
exists and is enforced; the work is converting the generic string error to a
typed error per AGENTS.md "Error Type Conventions".

**Note:** Phase 43 ARCH-05 will collapse the 4 stub architectures into a single
`StubArchitecture`. This plan establishes the typed error path now; Phase 43
inherits it during consolidation.
