## CLI Usage

```bash
# Throughput benchmark
cargo run -p vllm-lite-benchmarks --bin benchmark -- throughput --concurrency 8 --requests 100

# Latency benchmark
cargo run -p vllm-lite-benchmarks --bin benchmark -- latency --requests 1000 --warmup 5

# Combined benchmark with JSON output
cargo run -p vllm-lite-benchmarks --bin benchmark -- both --json
```
