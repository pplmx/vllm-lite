# Phase 15.5: Go K8s Operator - Plan

**Phase:** 15.5
**Created:** 2026-04-27
**Status:** Ready for execution

---

## Implementation Plan

### Task 1: Create Operator Structure

Create `k8s/operator/` with:
- `go.mod` - Go module definition
- `cmd/controller/main.go` - Entry point
- `pkg/api/v1/` - CRD types
- `pkg/controller/` - Reconciliation logic

### Task 2: Define vLLMEngine CRD

Create `config/crd/vllmengine-crd.yaml`:
- Model, replicas, image, GPU config
- Status with replicas, service URL
- Subresource status

### Task 3: Configure RBAC

Create `config/rbac/role.yaml`:
- ClusterRole with vllmengines, deployments, services permissions
- ClusterRoleBinding for controller

### Task 4: Update Helm Chart

Modify `k8s/charts/` to use Operator:
- Install CRD via Helm
- Deploy Operator instead of raw Deployment
