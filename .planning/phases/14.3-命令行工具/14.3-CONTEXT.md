# Phase 14.3: 命令行工具 - Context

**Gathered:** 2026-04-27
**Status:** Ready for planning

<domain>
## Phase Boundary

Implement CLI tools for model management and configuration validation.

</domain>

<decisions>
## Implementation Decisions

### CLI Architecture
- Separate `vllm` binary with subcommands
- `config validate` - Validate YAML config files
- `model list` - List models in directory
- `model info` - Show model metadata

### Config Validation
- Parse YAML with serde_yaml
- Check required fields: server, engine
- Validate port range (1-65535)
- Validate kv_blocks > 0

### Model Discovery
- Scan directories for model files
- Detect architecture from config.json
- Calculate directory sizes

</decisions>

### Reusable Assets

### Integration Points
