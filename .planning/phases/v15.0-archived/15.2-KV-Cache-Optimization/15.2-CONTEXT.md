---

## Implementation Decisions

### Chunked Prefill
- Split large prompts (seq_len > threshold) into chunks
- Process in chunks to avoid memory spikes
- Configurable chunk size (default: 512 tokens)

### FP8 KV Cache
- Use FP8 (E4M3) format for KV cache storage
- Fallback to FP16 for attention computation
- Accuracy validation against FP16 baseline

### Preemption Strategy
- When memory pressure exceeded, preempt running sequences
- Prioritize by remaining output length
- Graceful degradation under load
