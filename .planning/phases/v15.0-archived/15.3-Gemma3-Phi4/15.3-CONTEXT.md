## Phase Boundary

Support Google's Gemma3 and Microsoft's Phi-4 model architectures.

---

## Success Criteria

1. Gemma3 architecture registered with sliding window attention (SWA)
2. Gemma3-2B, 9B, 27B models load and generate tokens
3. Phi-4 architecture registered with NoPos (rotary) embedding
4. Phi-4-14B model loads and generates tokens

---

## Implementation Notes

### Gemma3 Architecture
- Uses Sliding Window Attention (SWA) for memory efficiency
- Grouped-Query Attention (GQA) for key-value heads
- RMSNorm with GeGLU activation
- Similar to Gemma2/Gemma4 patterns

### Phi-4 Architecture
- No positional embeddings (rotary only)
- SwiGLU MLP with partial rotation
- GQA support
- Similar to Phi-3 pattern
