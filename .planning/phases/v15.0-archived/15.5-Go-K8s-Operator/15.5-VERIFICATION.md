### Key Features

1. **CRD Specification**:
   - `model`: Model name to serve
   - `replicas`: Number of replicas (default: 1)
   - `image`: Container image
   - `gpuResource.count`: GPU count
   - `maxModelLen`: Maximum sequence length
   - `tensorParallelSize`: TP degree

2. **Reconciliation Logic**:
   - Creates/updates Deployment with vLLM server
   - Creates/updates Service for HTTP endpoint
   - Updates status with service URL
   - Handles scale changes via replicas field

3. **RBAC**:
   - ClusterRole with vllmengines, deployments, services permissions
   - ClusterRoleBinding for controller service account
