# Phase 13.1 Plan: K8s 基础

**Phase:** 13.1
**Created:** 2026-04-27
**Status:** Planned

## Task Breakdown

### Task 1: Health Check Endpoints (OPS-02, K8S-03)
- Add HTTP health endpoints to server (`/health/live`, `/health/ready`)
- Reference `crates/server/src/health.rs`
- Create liveness probe (always returns 200 when process running)
- Create readiness probe (returns 200 only when model loaded)

### Task 2: ConfigMap Integration (K8S-04)
- Update config loading to support ConfigMap paths
- Support `/etc/vllm/config.yaml` mounted path
- Add environment variable override for config path

### Task 3: Graceful Shutdown (K8S-05)
- Implement signal handling (SIGTERM, SIGINT)
- Drain in-flight requests before shutdown
- Configurable drain timeout

### Task 4: NodeMesh Extension (CLUSTER-01, CLUSTER-02)
- Extend `crates/dist/src/tensor_parallel/device_mesh.rs` to NodeMesh
- Add peer discovery via headless service DNS
- Implement DeviceList for multi-node

### Task 5: gRPC Transport Layer (CLUSTER-03)
- Add `tonic` and `prost` dependencies
- Create gRPC server for inter-node communication
- Implement ping/pong for health checking
- Add TLS support for gRPC

### Task 6: Helm Chart (K8S-01, OPS-01)
- Create `k8s/charts/vllm-lite/Chart.yaml`
- Create `k8s/charts/vllm-lite/values.yaml`
- Create templates for Deployment, Service, ConfigMap, HeadlessService
- Add PodDisruptionBudget
- Add deployment scripts

### Task 7: Kubernetes Operator (K8S-02)
- Create `k8s/operator/` Go module
- Define VLLMLite CRD
- Implement reconciliation loop
- Handle lifecycle (create, update, delete)

### Task 8: K8s Manifests
- Create `k8s/manifests/` with all resources
- Include RBAC (ServiceAccount, ClusterRole, ClusterRoleBinding)
- Include resource limits and requests
- Include NVIDIA GPU tolerations

## Dependencies

```
Health Endpoints → Graceful Shutdown
NodeMesh → gRPC Transport
Helm Chart → K8s Manifests
```
