<decisions>
## Implementation Decisions

### Verification Approach
- Single forward pass evaluates all draft tokens
- Token acceptance: accept if target_prob >= draft_prob (TokenLevel strategy)
- Early termination at first rejection

### KV Cache Management
- Accepted draft KV blocks copied/appended to target cache
- No recomputation of accepted prefixes
