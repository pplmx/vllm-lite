# Phase 40 Plan 03: Replace `Box<dyn Error>` with Typed `GrpcError`

**Goal:** Replace `Result<(), Box<dyn std::error::Error>>` in
`crates/dist/src/grpc.rs:129` (the `start_grpc_server` public function) with a
typed `GrpcError` enum (thiserror) per AGENTS.md convention. No `Box<dyn Error>`
remains in `crates/dist/src/` public API.
