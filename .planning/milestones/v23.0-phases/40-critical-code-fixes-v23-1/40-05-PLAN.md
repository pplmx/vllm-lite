# Phase 40 Plan 05: Implement `prefix_cache_hit_rate` Against Metrics Counters

**Goal:** Replace the `prefix_cache_hit_rate()` placeholder at
`crates/core/src/scheduler/engine.rs:555-559` (returns `0.0` always) with a real
implementation that reads `prefix_cache_hits` and `prefix_cache_requests`
counters from `EnhancedMetricsCollector`. Returns `hits / requests` (or `0.0`
when `requests == 0`).

**Preflight finding:** Both counters exist in `LockFreeMetrics` as
`Arc<AtomicU64>`, but `EnhancedMetricsCollector` exposes only the `record_*`
methods — no accessor for the values. This plan adds the accessors and wires
the scheduler through them.
