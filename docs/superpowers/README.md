# Superpowers Workspace

This directory is the **Superpowers** tool output path. Do not relocate or bulk-archive.

| Subdir | Purpose |
|--------|---------|
| [`specs/`](./specs/) | Design specs (`YYYY-MM-DD-topic-design.md`) |
| [`plans/`](./plans/) | Implementation plans (checkbox task lists) |

**Workflow:** Superpowers writes here → implement in code → distill durable decisions into [`docs/adr/`](../adr/) and [`CHANGELOG.md`](../../CHANGELOG.md).

Older shipped specs/plans stay in git for archaeology; they are not user-facing docs. See [`.planning/DOC-MAP.md`](../../.planning/DOC-MAP.md) for authority rules.
